### Implementation Notes for Devs

* **Typography:** All Headlines and Data Points must be uppercase. Data points (like the feature lists in Section E) should use the **Monospace** font stack.
* **Styling:** No soft shadows. Use `1px` borders (`#333` or `#E5E5EA` for active states).
* **Icons:** All icons must be monochrome (White/Silver). No colored backgrounds.
